# MVP Definition Prompt

Use this prompt to define the Minimum Viable Product scope.

---

## Context

> Briefly describe the project and the problem it solves.

## Core Questions

1. What is the single most important outcome for the first release?
2. Who is the primary user persona for the MVP?
3. What are the 3–5 features absolutely required for launch?
4. What features are explicitly **out of scope** for the MVP?
5. What does success look like at the end of the MVP phase?

## Output Format

- **MVP Goal:** One sentence describing the MVP objective.
- **In Scope:** Bullet list of included features.
- **Out of Scope:** Bullet list of explicitly excluded features.
- **Success Metrics:** 2–3 measurable KPIs.
- **Timeline Estimate:** Rough weeks/sprints estimate.
