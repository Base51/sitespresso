# Static Website Template

A minimal static website template with no build tooling required.

## Structure

```
static-website/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
└── images/
```

## Getting Started

1. Copy this folder to your new project.
2. Edit `index.html` as your entry point.
3. Add styles to `css/styles.css`.
4. Add scripts to `js/main.js`.
5. Place images in `images/`.

## Deployment

This site can be deployed to any static host:

- **GitHub Pages** — push to `gh-pages` branch or configure in repository Settings > Pages.
- **Azure Static Web Apps** — connect the repository and set the app location to this folder.
- **Netlify / Vercel** — drag and drop the folder or connect via Git.
